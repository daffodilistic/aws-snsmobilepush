Amazon SNS Mobile Push Android Mobile Push App
----------------------------------------------------
This sample is compatible with physical devices running android versions of 2.3 and later.
It is compatible with virtual devices running Google API 17 and later.

This sample creates an app to display messages from SNS Mobile Push services.

To use this sample:

1-  Make sure you have the latest android sdk, found here http://developer.android.com/sdk/index.html
	the google-play-services sdk, information about which can be found here
	http://developer.android.com/google/play-services/setup.html, and the android support library,
    which can be downloaded from the Android SDK Manager.
    ***Make sure all 3 are in your buildpath. However, make sure android.jar is NOT exported
       and the other 2 are exported.***
2- Follow the instructions found at http://docs.aws.amazon.com/sns/latest/dg/mobile-push-gcm.html .

For more information about Amazon SNS Mobile Push, please see docs.aws.amazon.com/sns/latest/dg/SNSMobilePush.html
